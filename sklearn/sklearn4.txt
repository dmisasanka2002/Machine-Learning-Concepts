# **Missing Data for Scikit-learn Overview**

---

## **1. Data Preprocessing (`sklearn.preprocessing`)**

| **Class/Function**                        | **Purpose**                                                   |
|-------------------------------------------|---------------------------------------------------------------|
| `sklearn.preprocessing.RobustScaler`      | Scales features using statistics that are robust to outliers.  |
| `sklearn.preprocessing.Imputer`           | (Deprecated) Imputation of missing values in the dataset.      |
| `sklearn.preprocessing.KBinsDiscretizer`  | Discretizes continuous features into discrete bins.           |

---

## **2. Model Selection and Hyperparameter Tuning (`sklearn.model_selection`)**

| **Class/Function**                             | **Purpose**                                                   |
|------------------------------------------------|---------------------------------------------------------------|
| `sklearn.model_selection.TimeSeriesSplit`      | Splits data for cross-validation, maintaining the temporal order of observations. |
| `sklearn.model_selection.PredefinedSplit`      | Performs cross-validation based on pre-defined train-test splits. |

---

## **3. Supervised Learning Algorithms**

### **3.1. Classification Models**

| **Class**                                      | **Purpose**                                                   |
|------------------------------------------------|---------------------------------------------------------------|
| `sklearn.linear_model.Perceptron`              | A simple linear classifier based on the perceptron learning algorithm. |
| `sklearn.svm.NuSVC`                           | Support Vector Classifier with a parameter for controlling the number of support vectors. |
| `sklearn.ensemble.AdaBoostClassifier`          | Adaptive boosting for combining weak learners into a strong classifier. |

### **3.2. Regression Models**

| **Class**                                      | **Purpose**                                                   |
|------------------------------------------------|---------------------------------------------------------------|
| `sklearn.linear_model.SGDRegressor`            | Implements linear regression using stochastic gradient descent. |
| `sklearn.kernel_ridge.KernelRidge`             | Regression model using kernel methods and ridge regularization. |
| `sklearn.gaussian_process.GaussianProcessRegressor` | Probabilistic regression using Gaussian processes.           |

---

## **4. Unsupervised Learning Algorithms**

### **4.1. Clustering**

| **Class**                                      | **Purpose**                                                   |
|------------------------------------------------|---------------------------------------------------------------|
| `sklearn.cluster.DBSCAN`                       | A density-based clustering algorithm that can detect outliers. |
| `sklearn.cluster.MiniBatchKMeans`              | A variation of KMeans for large datasets that uses mini-batches. |
| `sklearn.mixture.GaussianMixture`              | Gaussian Mixture Models (GMM) for clustering based on probability. |

### **4.2. Dimensionality Reduction**

| **Class**                                      | **Purpose**                                                   |
|------------------------------------------------|---------------------------------------------------------------|
| `sklearn.decomposition.FASTICA`                | Fast Independent Component Analysis for dimensionality reduction. |
| `sklearn.decomposition.LatentDirichletAllocation` | Topic modeling using LDA for unsupervised learning in natural language processing. |

---

## **5. Feature Engineering and Selection**

| **Class/Function**                             | **Purpose**                                                   |
|------------------------------------------------|---------------------------------------------------------------|
| `sklearn.feature_selection.SelectFpr`          | Selects features based on a specified false positive rate.    |
| `sklearn.feature_selection.SelectFromModel`    | Selects features based on their importance scores from a trained model. |

---

## **6. Text and Natural Language Processing (NLP)**

| **Class**                                      | **Purpose**                                                   |
|------------------------------------------------|---------------------------------------------------------------|
| `sklearn.feature_extraction.text.TfidfTransformer` | Transforms the term-frequency matrix into a TF-IDF (Term Frequency-Inverse Document Frequency) matrix. |

---

## **7. Pipelines and Workflows (`sklearn.pipeline`)**

| **Class/Function**                             | **Purpose**                                                   |
|------------------------------------------------|---------------------------------------------------------------|
| `sklearn.pipeline.make_pipeline`               | Helper function to create pipelines from a list of steps.     |
| `sklearn.pipeline.Pipeline`                   | A wrapper to chain preprocessing, feature extraction, and model fitting in a single workflow. |

---

## **8. Model Evaluation and Metrics (`sklearn.metrics`)**

### **8.1. Classification Metrics**

| **Function**                                   | **Purpose**                                                   |
|------------------------------------------------|---------------------------------------------------------------|
| `sklearn.metrics.average_precision_score`       | Computes average precision (AP) score for classification tasks. |
| `sklearn.metrics.balanced_accuracy_score`      | Computes balanced accuracy score, useful for imbalanced datasets. |
| `sklearn.metrics.log_loss`                     | Computes the log loss (logarithmic loss) for probabilistic predictions. |
| `sklearn.metrics.roc_curve`                    | Computes Receiver Operating Characteristic curve for binary classification tasks. |

### **8.2. Regression Metrics**

| **Function**                                   | **Purpose**                                                   |
|------------------------------------------------|---------------------------------------------------------------|
| `sklearn.metrics.median_absolute_error`         | Computes the median of the absolute errors between predictions and true values. |
| `sklearn.metrics.max_error`                    | Computes the maximum residual error in regression.            |

---

## **9. Advanced Topics**

### **9.1. Model Interpretability (`sklearn.inspection`)**

| **Function**                                   | **Purpose**                                                   |
|------------------------------------------------|---------------------------------------------------------------|
| `sklearn.inspection.plot_partial_dependence`   | Plots partial dependence of features for better model interpretability. |

### **9.2. Anomaly Detection**

| **Class**                                      | **Purpose**                                                   |
|------------------------------------------------|---------------------------------------------------------------|
| `sklearn.ensemble.IsolationForest`             | Detects anomalies using isolation forests, works well with high-dimensional data. |
| `sklearn.covariance.EllipticEnvelope`          | A robust method for outlier detection based on Gaussian distribution fitting. |
